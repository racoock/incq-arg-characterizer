# Example Output
This directory will contain the output files generated by the workflow.

- `incQ_hits.tsv`: Table of IncQ replicon hits (contig, start, end, score, etc.)
- `arg_hits.tsv`: Table of ARG hits (contig, gene, identity, coverage)
- `mge_context.tsv`: Table of MGEs found near ARGs

Run `./run_analysis.sh` to generate actual results (placeholder scripts produce simulated data).
